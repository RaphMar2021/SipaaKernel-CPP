Enumerator
VFS_SUCCESS 

Returned when succeded.

VFS_NULL_NODE 

Returned when the provided FS node is NULL.

VFS_SLOTS_FULL 

Returned when the mountpoint list is full.

VFS_SLOT_ALREADY_TAKEN 

Returned when a filesystem is already mounted on the provided path.

VFS_CANNOT_UNMOUNT_ROOT 

Returned when trying to unmount the filesystem.

VFS_NOT_MOUNTED 

Returned when trying to unmount an inexistant mount point.

VFS_NOT_ALLOWED

Returned when a VFS operation is not supported or implemented.
